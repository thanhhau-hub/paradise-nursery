# Paradise Nursery Shopping Application

## Project Name: Paradise Nursery

A beautiful and functional e-commerce plant shop application built with React and Redux.

## Description

Paradise Nursery is a modern plant shopping application where users can browse through a variety of plants, add them to their shopping cart, and manage their purchases. The application features a clean, user-friendly interface with full cart management capabilities.

## Features

- **Landing Page**: Welcome page with company information and call-to-action
- **Product Listing**: Display of available plants with images, descriptions, and prices
- **Shopping Cart**: Full cart management with add, remove, and quantity update features
- **Redux State Management**: Efficient state management using Redux Toolkit
- **Responsive Design**: Mobile-friendly interface

## Technologies Used

- React.js
- Redux Toolkit
- React Router
- CSS3
- JavaScript ES6+

## Installation

```bash
npm install
npm start
```
